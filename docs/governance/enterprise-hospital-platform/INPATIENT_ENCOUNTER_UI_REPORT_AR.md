# تقرير واجهة مستخدم التنويم الداخلي (Inpatient)

يوثق تفاصيل واجهة وتصميم عمليات التنويم الداخلي للمرضى بالمنصة.

## عناصر الواجهة المطبقة
1. **حالة التنويم وتخصيص الأسرة**: بطاقات توضح القسم الداخلي، رقم الجناح، والسرير المخصص للمريض.
2. **خطة الرعاية والتدخلات التمريضية**: لوحة شاملة توضح الأدوية المجدولة والتدخلات اليومية.
3. **مسار التنويم (Inpatient Timeline)**: يعرض تسلسل مراحل التنويم من طلب الدخول حتى التنسيق للخروج من المستشفى.
