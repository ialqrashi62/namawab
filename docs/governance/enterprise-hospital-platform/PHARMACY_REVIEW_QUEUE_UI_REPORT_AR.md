# تقرير واجهة مستخدم صندوق مراجعة الصيدلي (Pharmacy Review Queue)

يوثق تفاصيل واجهة وتصميم صندوق مراجعة الصيدلي بالمنصة.

## عناصر الواجهة المطبقة
1. **صندوق الطلبات الواردة**: عرض قائمة الأدوية المطلوبة للزيارة الحالية ومستوى خطورتها وحالتها.
2. **شارات التحقق والأمان**: إظهار شارات تدل على الحاجة لمراجعة الصيدلي أو تفعيل بروتوكول الحساسية.
