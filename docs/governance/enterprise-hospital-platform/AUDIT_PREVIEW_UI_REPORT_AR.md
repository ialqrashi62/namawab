# تقرير واجهة مستخدم معاينة سجل التدقيق (Audit Preview UI)

يوثق تفاصيل واجهة وتصميم لوحة معاينة سجل التدقيق الأمني بالمنصة.

## عناصر الواجهة المطبقة
1. **نموذج اختيار الدور النشط**: واجهة تفاعلية تتيح تبديل الدور الوظيفي للمستخدم (طبيب، صيدلي، موظف فوترة).
2. **شاشة عرض حدث التدقيق**: نافذة برمجية تعرض تفاصيل الحدث الأمني المولد بصيغة JSON حركية عند محاولة تنفيذ أي إجراء.
