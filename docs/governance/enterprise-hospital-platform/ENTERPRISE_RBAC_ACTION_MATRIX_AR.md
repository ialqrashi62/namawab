# مصفوفة صلاحيات الوصول المستندة إلى الأدوار (RBAC Action Matrix)

توثق هذه المصفوفة الصلاحيات المتاحة لكل دور وظيفي بالمنصة والقيود المفروضة على العمليات الحساسة.

## جدول الصلاحيات (Role-Action Matrix)
| الدور الوظيفي | معاينة المنشأة | كتابة الملاحظات | مراجعة الصيدلية | الفوترة والمطالبات | الصرف النهائي |
| :--- | :---: | :---: | :---: | :---: | :---: |
| **طبيب (Physician)** | مسموح | مسموح (مسودة) | محظور | محظور | محظور |
| **صيدلي (Pharmacist)** | مسموح | محظور | مسموح (معاينة) | محظور | محظور |
| **موظف فوترة (Billing)** | مسموح | محظور | محظور | مسموح (معاينة) | محظور |
| **مدير النظام (Admin)** | مسموح | مسموح (مسودة) | مسموح (معاينة) | مسموح (معاينة) | محظور |
| **مدقق (Auditor)** | مسموح | محظور | مسموح (معاينة) | مسموح (معاينة) | محظور |

*ملاحظة: جميع العمليات النهائية (Final Actions) محظورة بالكامل لجميع الأدوار في هذه المرحلة.*
