# مصفوفة تصنيف البيانات ونطاق الوصول (Data Classification Matrix)

توثق هذه المصفوفة مستويات سرية البيانات المطبقة بالمنصة والقيود المفروضة على تداولها.

## تصنيف البيانات
1. **بيانات طبية سرية للغاية (High PHI)**: تشمل نتائج الفحوصات والملاحظات الطبية والأوامر السريرية (تتطلب حماية RLS قصوى وتوقيعاً معتمداً).
2. **بيانات مالية حساسة (Financial Data)**: تشمل الفواتير ومسودات مطالبات التأمين ونسب التحمل (تقتصر على موظفي الفوترة والجودة).
3. **بيانات تشغيلية عامة (Operational Data)**: تشمل قائمة الأقسام والمنشآت ومواعيد العمل (متاحة لجميع الكوادر الطبية والإدارية).
