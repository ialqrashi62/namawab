# تصميم مسارات العمل العيادات الخارجية والتنويم (OPD & Inpatient)

يوضح هذا المستند المخطط التدفقي لمسار زيارات العيادات الخارجية (OPD) وعمليات التنويم الداخلي (Inpatient).

## مسار العيادات الخارجية (OPD Flow)
1. **تسجيل المريض / فرز الحالات**: حجز موعد أو زيارة عاجلة وتسجيل العلامات الحيوية.
2. **معاينة الطبيب**: الفحص السريري وتدوين التشخيص وطلب الفحوصات أو الأدوية.
3. **الخروج والمتابعة**: طباعة الوصفة الطبية، حجز موعد المتابعة، وإغلاق ملف الزيارة.

## مسار التنويم الداخلي (Inpatient Flow)
1. **طلب التنويم**: طلب إدخال المريض للقسم الداخلي من قبل الطبيب المعالج.
2. **تخصيص السرير**: اختيار الغرفة والسرير المتاحين بالقسم المعني وتثبيتهما للمريض.
3. **جولات الأطباء والرعاية التمريضية**: جولات المرور الطبي اليومية، تحديث خطة الرعاية، وإدارة الطلبات والنتائج.
4. **خطة الخروج**: إعداد ملخص الخروج الطبي والتنسيق مع الرعاية المنزلية أو العيادات الخارجية لإتمام العلاج.
