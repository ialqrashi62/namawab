# خطة تنفيذ مسارات الطلبات الطبية السريرية

تصف خطة دمج وتفعيل مساحات عمل الطلبات الطبية والتحقق من صلاحياتها بالمشروع.

## تفاصيل التنفيذ
1. **الملف المستقل للطلبات**: تجميع منطق وعمليات الكتالوج الطبي داخل الملف `public/js/clinical-orders.js`.
2. **عزل الصلاحيات**: التحقق من نوع المنشأة الطبية لمنع طلب الإجراءات المتقدمة في الوحدات الصحية ومراكز الرعاية الأولية.
3. **دمج الواجهات**: ربط لوحة منشئ الطلبات الطبية حركياً ببيئة عمل الزيارة السريرية النشطة للمريض.
