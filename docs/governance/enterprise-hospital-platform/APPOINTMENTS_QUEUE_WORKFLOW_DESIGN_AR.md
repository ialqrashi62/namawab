# تصميم مسارات تدفق المواعيد وطوابير انتظار المرضى

يوضح هذا المستند المخطط التدفقي لمسار حجز المواعيد ودخول المرضى والفرز الطبي.

## مسار حجز المواعيد (Appointment Booking Flow)
1. **اختيار المنشأة والقسم**: يحدد المستخدم المنشأة المعنية والقسم الطبي المطلوبة زيارته.
2. **تحديد نوع الزيارة**: اختيار موعد مجدول، حضور عاجل (Walk-in)، أو إحالة داخلية.
3. **عرض المواعيد المتاحة**: إظهار الفترات الزمنية المتاحة محاكاةً وحجز الموعد كمسودة مع حظر أي كتابة في قاعدة البيانات.

## مسار طابور الانتظار والفرز (Patient Queue & Triage Flow)
1. **وصول المريض (Check-in)**: تسجيل حضور المريض وإضافته لطابور الانتظار بالقسم.
2. **الفرز الطبي (Triage)**: قيام الممرض بقياس العلامات الحيوية وتحديد أولوية الحالة (حالة عاجلة تتطلب الدخول الفوري، أو حالة عادية).
3. **بدء المعاينة (Encounter Launch)**: استدعاء المريض لغرفة المعاينة وبدء الزيارة السريرية للطبيب.
