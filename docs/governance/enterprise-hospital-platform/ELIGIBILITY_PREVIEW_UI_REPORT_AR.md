# تقرير واجهة مستخدم التحقق من الأهلية (Eligibility Preview)

يوثق تفاصيل واجهة وتصميم شاشة التحقق من أهلية التأمين للمريض.

## عناصر الواجهة المطبقة
1. **نموذج التحقق من التأمين**: واجهة تتيح اختيار جهة التأمين والتحقق التفاعلي من نشاط التغطية.
2. **تفاصيل التغطية المالية**: عرض نسب التحمل للمريض والحد الأقصى للتغطية بشكل تفاعلي محاكى.
