# تقرير واجهة مستخدم مسودات المطالبات (Claim Draft Preview)

يوثق تفاصيل واجهة وتصميم شاشة مسودات مطالبات التأمين بالمنصة.

## عناصر الواجهة المطبقة
1. **عرض تفاصيل المطالبة**: حصر القيمة الإجمالية للمطالبة وعرض حالتها كمسودة غير مرسلة.
2. **تنبيهات التحقق من المطالبات**: شارات توضح الأخطاء البرمجية الشائعة التي تتطلب التعديل (مثل نقص التوقيع أو البيانات الناقصة).
