# تقرير إعداد وتجهيز عقود الواجهات البرمجية (API Contracts)

يوثق هذا التقرير تفاصيل عقود الواجهات البرمجية وتصميم الـ DTOs والـ OpenAPI للمرحلة القادمة.

## تفاصيل العقود والمواصفات
1. **إصدار العقد**: تم اعتماد الإصدار التجريبي الأول للمواصفات (`1.0.0-draft`).
2. **الـ DTOs المحددة**: تعريف هياكل البيانات لجميع العناصر الأساسية (المنشآت، الزيارات، الطلبات، الصيدلية، الفوترة).
3. **جاهزية OpenAPI**: كتابة المواصفات الفنية لجميع الواجهات المطلوبة بصيغة YAML تمهيداً للربط الفعلي.
