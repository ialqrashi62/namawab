# تقرير واجهة مستخدم طلبات الأدوية والوصفات (Medication)

يوثق تفاصيل واجهة وتصميم الوصفات الدوائية بالمنصة.

## عناصر الواجهة المطبقة
1. **قائمة الأدوية المتاحة**: اختيار الدواء المناسب من الكتالوج الطبي المحاكى.
2. **فحص التعارض والحساسية**: نافذة تحذيرية منبثقة تظهر تلقائياً عند إضافة أي دواء لتنبيه الطبيب المعالج.
