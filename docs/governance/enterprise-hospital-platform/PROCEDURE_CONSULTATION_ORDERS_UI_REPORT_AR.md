# تقرير واجهة مستخدم طلبات الإجراءات والاستشارات (Procedures & Consultations)

يوثق تفاصيل واجهة وتصميم طلبات الإجراءات الطبية والاستشارات بين الأقسام.

## عناصر الواجهة المطبقة
1. **طلبات الإجراءات الطبية**: قائمة الإجراءات الطبية الموصى بها مع تحديد متطلبات إقرار موافقة المريض.
2. **طلبات الاستشارة بين الأقسام**: نموذج توجيه المريض لقسم طبي آخر لأخذ الاستشارة الطبية (مثل استشارة القلب أو المخ والأعصاب).
