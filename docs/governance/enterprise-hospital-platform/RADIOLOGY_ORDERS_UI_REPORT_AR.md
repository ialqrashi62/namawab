# تقرير واجهة مستخدم طلبات الأشعة (Radiology)

يوثق تفاصيل واجهة وتصميم طلبات الفحوصات الإشعاعية بالمنصة.

## عناصر الواجهة المطبقة
1. **كتالوج الأشعة المتاحة**: قائمة الفحوصات الإشعاعية المخصصة للقسم (مثل الأشعة السينية والرنين المغناطيسي).
2. **مستوى الخطورة والتحذيرات**: إظهار إشارات تحذيرية للفحوصات التي تتطلب رعاية خاصة أو صبغة ملونة.
