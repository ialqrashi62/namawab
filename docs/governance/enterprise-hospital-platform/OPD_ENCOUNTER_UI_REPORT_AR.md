# تقرير واجهة مستخدم زيارات العيادات الخارجية (OPD)

يوثق تفاصيل واجهة وتصميم زيارات العيادات الخارجية.

## عناصر الواجهة المطبقة
1. **التقييم التمريضي**: نموذج إدخال العلامات الحيوية والفرز الطبي مع أزرار الحفظ التجريبي.
2. **معاينة الطبيب (SOAP)**: مربعات تدوين المعاينة والتشخيص السريري للمريض.
3. **مسار الزيارة (OPD Timeline)**: مؤشر مرئي يوضح تقدم المريض خلال مراحل الزيارة الحالية.
