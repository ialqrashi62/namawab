# تقرير واجهة مستخدم معاينة مخزون FEFO (FEFO Inventory Preview)

يوثق تفاصيل واجهة وتصميم شاشة معاينة دفعات المخزون وترتيب الصلاحية.

## عناصر الواجهة المطبقة
1. **جدول دفعات المخزون**: عرض كود الدفعة، تاريخ الانتهاء، الكمية المتاحة، والترتيب الصلاحي (FEFO Rank).
2. **أزرار المعاينة والصرف التجريبي**: تفعيل زر عرض الدفعات ومعاينة إذن الصرف مع إبقاء زر الصرف النهائي معطلاً.
