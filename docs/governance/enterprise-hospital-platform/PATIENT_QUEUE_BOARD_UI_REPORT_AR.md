# تقرير واجهة مستخدم لوحة طابور المرضى (Patient Queue Board)

يوثق تفاصيل واجهة وتصميم لوحة طابور انتظار المرضى بالقسم الطبي.

## عناصر الواجهة المطبقة
1. **جدول طابور الانتظار**: عرض رقم الانتظار، نوع الزيارة، أولوية الفرز، ووقت الانتظار المتوقع للمريض.
2. **شارات الحالة والأولوية**: إظهار بطاقات مرئية توضح حالة الانتظار (بانتظار الفرز، بانتظار العيادة) ومستوى أولوية الحالة.
