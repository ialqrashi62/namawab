# تقرير مراجعة القدرات الحالية لملفات المرضى والزيارات السريرية

يوثق هذا التقرير تحليل المسارات والواجهات والعمليات الحالية لإدارة المرضى والزيارات الطبية.

## القدرات الحالية في المنصة
1. **إدارة ملفات المرضى (Patient Demographics)**: تتيح المنصة تسجيل المرضى وإدخال بياناتهم الشخصية وتوليد رقم الملف الطبي (MRN) الخاص بكل مريض.
2. **شاشة العلامات الحيوية (Nursing Vitals)**: تتوفر شاشات مخصصة للممرضين لإدخال الضغط والنبض والحرارة ومستوى الألم للمريض.
3. **لوحة الطبيب ومعاينته (SOAP Notes)**: يدعم النظام تدوين الملاحظات الطبية بأسلوب SOAP المعياري وقائمة التشخيصات النشطة.
4. **طلب الفحوصات والنتائج (Lab & Radiology Orders)**: ترتبط المنصة كلياً بمسارات طلبات المختبر والأشعة وخطوط الصيدلية السريرية.
