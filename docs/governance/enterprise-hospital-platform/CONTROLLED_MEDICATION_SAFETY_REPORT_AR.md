# تقرير أمان وجودة صرف الأدوية الخاضعة للرقابة

يوثق المعايير والضوابط المطبقة للتعامل الآمن مع الأدوية المخدرة والخاضعة للرقابة بالمنصة.

## معايير الأمان المطبقة
1. **تحديد الأدوية الخاضعة للرقابة**: ربط الأدوية الحساسة (مثل المورفين) بشارات تحذيرية حمراء واضحة للعيان.
2. **إلزامية التحقق البشري المزدوج**: إظهار متطلبات التوقيع المزدوج للطبيب والصيدلي قبل صرف الأدوية المخدرة محاكاةً.
