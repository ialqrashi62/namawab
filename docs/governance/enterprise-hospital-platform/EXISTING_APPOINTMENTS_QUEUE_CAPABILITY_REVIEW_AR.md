# تقرير مراجعة القدرات الحالية للمواعيد وطوابير المرضى والفرز

يوثق هذا التقرير تحليل المسارات والواجهات والعمليات الحالية لحجز المواعيد وإدارة لوحة انتظار المرضى وإجراء الفرس الطبي والفرز.

## القدرات الحالية في المنصة
1. **لوحة حجز المواعيد (Appointment Booking)**: شاشة مخصصة تتيح اختيار القسم ونوع الزيارة وعرض المواعيد المتاحة محاكاةً.
2. **طابور انتظار المرضى (Patient Queue Board)**: لوحة تفاعلية تعرض قائمة المرضى المنتظرين بالقسم، أوقات الانتظار، والأولويات الطبية.
3. **الفرز وتحديد الأولويات (Triage & Priority)**: دعم تحديد درجة خطورة الحالات (عاجل، عادي) وربطها ببطاقات الفرز التمريضي.
4. **التوجيه بين الأقسام (Department Routing)**: إمكانية تحويل وتوجيه المرضى بين الأقسام الطبية كإحالات داخلية.
