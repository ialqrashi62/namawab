# تصميم مسارات تدفق نتائج الفحوصات والتوثيق السريري

يوضح هذا المستند المخطط التدفقي لمسار نتائج الفحوصات والتوثيق السريري للأقسام الطبية.

## مسار نتائج الفحوصات (Results Flow)
1. **استلام النتائج**: يتم عرض النتائج الواردة من المختبر أو الأشعة في صندوق النتائج (Results Inbox).
2. **تحديد الحالات الحرجة**: فحص محاكى للنتائج وتحديد ما إذا كانت النتيجة غير طبيعية (Abnormal) وعرض إشارة تنبيهية.
3. **اعتماد الطبيب**: يقوم الطبيب بمراجعة النتيجة وتدوين ملاحظاته السريرية دون تفعيل أي تشخيص نهائي أو خطة علاجية آلية.

## مسار التوثيق السريري (Clinical Documentation Flow)
1. **فتح ملف التوثيق**: فتح مسودة الملاحظة الطبية للزيارة (Clinical Note).
2. **تدوين نموذج SOAP**: كتابة التفاصيل السريرية (الشكوى، الفحص، التقييم، الخطة).
3. **حفظ المسودة**: حفظ الملاحظة الطبية كمسودة غير نهائية (Draft Only) دون إمكانية التوقيع النهائي لضمان المراجعة الإدارية والطبية المزدوجة.
