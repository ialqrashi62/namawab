# خطة تنفيذ مسارات الصيدلية السريرية ومخزون FEFO

تصف خطة دمج وتفعيل مساحات عمل الصيدلية والمخزون الطبي والتحقق من صلاحياتها بالمشروع.

## تفاصيل التنفيذ
1. **الملف المستقل للصيدلية**: تجميع منطق وعمليات الصيدلية والمخزون داخل الملف `public/js/clinical-pharmacy.js`.
2. **دمج لوحة المراجعة**: ربط لوحة `Clinical Pharmacy & FEFO` حركياً ببيئة عمل الزيارة السريرية النشطة للمريض.
3. **ترتيب الصلاحية المطور**: تفعيل خوارزمية ترتيب الدفعات حسب تاريخ الانتهاء وعرضها في جداول تفاعلية محاكاة.
