# مسودة نموذج البيانات المقترح للميزات والواجهات الجديدة (بدون DDL)

يوضح هذا المستند التصميم الهيكلي المقترح للجداول وقواعد البيانات المستقبلية دون إجراء أي تعديل فعلي على قاعدة البيانات الحالية.

## الجداول المقترحة وتصميمها
1. **جدول المنشآت (facilities)**: لتخزين بيانات المستشفيات والعيادات ونوع التراخيص.
2. **جدول الزيارات (encounters)**: لربط المرضى بالأطباء وتوثيق نوع الزيارة وحالة القبول.
3. **جدول دفعات المخزون (inventory_batches)**: لتتبع أرقام الدفعات وتواريخ انتهاء الصلاحية وخصائص ترتيب FEFO.
4. **جدول مطالبات التأمين (claim_drafts)**: لحفظ مسودات الفواتير والمطالبات وبيانات الربط مع نفيس.
