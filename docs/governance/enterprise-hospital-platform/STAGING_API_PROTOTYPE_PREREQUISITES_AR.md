# متطلبات ومستلزمات بيئة الفحص والتكامل التجريبية (Staging Prerequisites)

توثق هذه المتطلبات الشروط الفنية اللازمة لتهيئة بيئة الفحص Staging لتجربة الربط التجريبي للواجهات.

## متطلبات بيئة الفحص
1. **توفير خادم مستقل لبيئة الفحص**: معزول تماماً عن خادم الإنتاج الفعلي.
2. **قاعدة بيانات فحص محاكاتية**: تحتوي على بيانات تجريبية خالية تماماً من أي معلومات حقيقية للمرضى.
3. **الاتصال مع بيئة فحص نفيس (NPHIES Sandbox)**: لتهيئة واختبار إرسال واستقبال المطالبات التأمينية.
