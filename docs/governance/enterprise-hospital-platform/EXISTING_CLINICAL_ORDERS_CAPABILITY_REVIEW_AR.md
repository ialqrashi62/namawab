# تقرير مراجعة القدرات الحالية للطلبات الطبية والوصفات الصيدلانية

يوثق هذا التقرير تحليل المسارات والواجهات والعمليات الحالية لإصدار وإدارة الطلبات الطبية (المختبر، الأشعة، الأدوية، الإجراءات، الاستشارات).

## القدرات الحالية في المنصة
1. **طلبات المختبر (Lab Orders)**: توفر شاشات مخصصة لإدخال طلبات الفحوصات المخبرية وعرض قائمة الفحوصات المتاحة بالقسم الطبي.
2. **طلبات الأشعة (Radiology Orders)**: يدعم النظام طلب الفحوصات الإشعاعية وربط صور الأشعة بملف المريض الطبي.
3. **الوصفات الدوائية (Prescriptions)**: تتيح الصيدلية السريرية إعداد مسودات الوصفات الدوائية ومراجعتها قبل الصرف.
4. **الإجراءات والاستشارات (Procedures & Consultations)**: تدعم المنصة تدوين طلبات الاستشارة بين الأقسام والإجراءات الطبية الموصى بها كمسودات.
