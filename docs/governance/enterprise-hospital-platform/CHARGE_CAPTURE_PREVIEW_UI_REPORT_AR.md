# تقرير واجهة مستخدم تسجيل الرسوم (Charge Capture Preview)

يوثق تفاصيل واجهة وتصميم لوحة تسجيل الرسوم الطبية بالمنصة.

## عناصر الواجهة المطبقة
1. **لوحة عرض الرسوم**: عرض تفاعلي للخدمات الطبية المضافة وقيمها المالية (مثل رسوم الاستشارة وصورة الدم).
2. **تحذيرات نقص التوثيق**: إظهار تنبيهات واضحة للمستخدم في حال وجود خدمات غير موثقة طبياً بالملف السريري.
