# تقرير واجهة مستخدم حجز المواعيد (Appointment Booking Preview)

يوثق تفاصيل واجهة وتصميم نموذج حجز المواعيد بالمنصة.

## عناصر الواجهة المطبقة
1. **نموذج اختيار الموعد**: واجهة تتيح للطبيب أو الموظف اختيار نوع الزيارة والتاريخ المفضل محاكاةً.
2. **أزرار الحجز التجريبية**: تعطيل أزرار التأكيد وإبقاء الحجز في حالة مسودة غير نهائية (Draft).
