# تقرير واجهة مستخدم طلبات المختبر (Lab)

يوثق تفاصيل واجهة وتصميم طلبات المختبر بالمنصة.

## عناصر الواجهة المطبقة
1. **كتالوج الفحوصات المتاحة**: قائمة منسدلة تعرض فحوصات المختبر المتاحة بالقسم حركياً (مثل صورة الدم الكاملة ووظائف الكبد).
2. **حالة الطلب**: إظهار الطلب المضاف في قائمة المسودات وتحديد حالة الفحص كمعلق أو بانتظار سحب العينة.
