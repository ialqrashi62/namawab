# خطة تنفيذ مسارات المواعيد وطوابير المرضى

تصف خطة دمج وتفعيل مساحات عمل المواعيد والانتظار والتحقق من صلاحياتها بالمشروع.

## تفاصيل التنفيذ
1. **الملف المستقل للمواعيد**: تجميع منطق وعمليات طوابير الانتظار داخل الملف `public/js/appointments-queue.js`.
2. **لوحة الطابور الموحدة**: ربط جدول `Patient Queue Board` حركياً ببيئة عمل القسم الطبي المعني لعرض الحالات المنتظرة.
3. **حماية مسارات العمل**: منع تفعيل طلبات التنويم الداخلي (Inpatient Admissions) في مراكز الرعاية الأولية والعيادات الخارجية وتحديدها للمستشفيات فقط.
