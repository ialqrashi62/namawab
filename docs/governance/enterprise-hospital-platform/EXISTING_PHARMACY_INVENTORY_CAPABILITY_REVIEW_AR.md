# تقرير مراجعة القدرات الحالية للصيدلية والمخزون الطبي

يوثق هذا التقرير تحليل المسارات والواجهات والعمليات الحالية للصيدلية السريرية وإدارة المخزون والتبديل الدوائي والتحذيرات.

## القدرات الحالية في المنصة
1. **إدارة الوصفات والطلبات الدوائية (Prescriptions)**: إمكانية تدوين الأدوية المطلوبة وربطها بالزيارة السريرية النشطة للمريض.
2. **تنبيهات سلامة الدواء (Medication Safety)**: فحص حساسية الأدوية وحظر تداول المواد المخدرة أو الخاضعة للرقابة دون تدقيق.
3. **تتبع صلاحية المخزون (FEFO Inventory)**: تصفية وترتيب دفعات الأدوية المخزنة بناءً على تاريخ انتهاء الصلاحية الأقرب.
4. **شاشة المعاينة والصرف (Dispense Preview)**: توفير مسودة إذن صرف لمراجعة الدفعة والكمية والتحقق منها قبل أي حركة مخزنية.
