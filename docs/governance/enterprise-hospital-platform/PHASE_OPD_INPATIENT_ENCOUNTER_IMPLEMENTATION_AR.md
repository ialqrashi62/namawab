# خطة تنفيذ مسارات العيادات الخارجية والتنويم

تصف خطة دمج وتفعيل بيئة العمل الموحدة للزيارات الطبية والتنويم الداخلي بالمشروع.

## تفاصيل التنفيذ
1. **الملف المستقل للزيارات**: تجميع منطق وعمليات الزيارات الطبية داخل الملف `public/js/encounter-workflow.js`.
2. **عزل المسارات**: التحقق من صلاحية المنشأة الحالية قبل تفعيل أي زيارة تنويم داخلي لحماية مسارات العمل من الأخطاء التشغيلية.
3. **واجهة تفاعلية موحدة**: دمج شاشة الزيارة السريرية `renderEncounterWorkspace` لتفتح حركياً داخل بيئة عمل القسم الطبي المعني.
